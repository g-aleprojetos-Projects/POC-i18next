# POC - i18Next

## Setup
 - Visual studio Code

 ## Framework
  - React typescipt

### Faça um fork do projeto
### Instalar as bibliotecas
```bash
yarn install
```

### Rodar o projeto
```bash
yarn start
```

### Rodar os testes
```bash
yarn test
```

### Rodar os testes e emite o reatório coverage
```bash
yarn run test --coverage
```
